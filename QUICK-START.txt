🚀 GET STARTED IN 3 EASY STEPS:

1. OPEN THE APPLICATION
   ────────────────────────────────────────────────────────────────────────
   • Double-click "index.html" to open in your default browser
   • OR right-click "index.html" → Open With → Choose your browser
   
   ✓ Works on: Chrome, Firefox, Safari, Edge (latest versions)
   ✓ No installation or server required!


2. ADD IBM LOGO (OPTIONAL)
   ────────────────────────────────────────────────────────────────────────
   • Download IBM logo from IBM Brand Center or Marketing Resources
   • Save as "ibm-logo.svg" in the "assets" folder
   • See "assets/IBM-LOGO-INSTRUCTIONS.txt" for detailed steps
   
   📌 The app works perfectly without the logo - it will be hidden automatically


3. CREATE YOUR FIRST USE CASE
   ────────────────────────────────────────────────────────────────────────
   • Click the "New Use Case" button
   • Fill in the required information:
     - Use Case Name
     - Scope
     - Benefits  
     - Outputs
     - Team Member Emails
   • Add optional comments/notes
   • Click "Save Use Case"


📖 WHAT YOU CAN DO:

   ✓ Submit new watsonx challenge ideas
   ✓ Track team members and their contributions
   ✓ Search and filter all use cases
   ✓ Add team member photos, names, and titles
   ✓ Export/import data to share with teammates
   ✓ Edit and delete use cases


💾 DATA STORAGE:

   • All data is stored locally in your browser
   • Data persists between sessions
   • To share with teammates:
     - Use Export/Import buttons, OR
     - Share the entire "website files" folder


📚 NEED MORE HELP?

   • All files are self-contained - no internet connection needed!
